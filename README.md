Este projeto foi desenvolvido como um exercício prático de lógica de programação, focado na estrutura de dados fundamental do JavaScript: **Arrays de Objetos**.

## 📝 O Desafio

O objetivo era criar uma lista de alunos e demonstrar o conhecimento em:
- Criação de objetos com múltiplas propriedades.
- Armazenamento de objetos dentro de um Array.
- Acesso a elementos específicos via índice.
- Manipulação dinâmica usando a propriedade `.length`.

### Requisitos do Desafio:
- [x] Criar um array chamado `alunos` com pelo menos 5 objetos.
- [x] Cada objeto deve conter: `nome`, `idade` e `disciplinaPreferida`.
- [x] Acessar o nome do **primeiro** aluno.
- [x] Acessar a disciplina preferida do **último** aluno.
